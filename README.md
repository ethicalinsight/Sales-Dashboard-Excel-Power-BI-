# Sales Dashboard

## Overview
This project demonstrates the creation of an interactive Sales Dashboard using Power BI.  
The dashboard provides insights into  Regional Sales Performance, Top 5 Model, and Sales Trends.

## Features
- Data preprocessing in Excel (cleaning, formatting, pivot tables).
- Import into Power BI for visualization.
- Interactive charts: bar, line, and column chart visuals.
- KPIs for revenue, quantity sold, and top-performing products.

## How to Use
1. Open the `.pbix` file in Power BI Desktop.
2. Explore visuals using slicers for product category, region, and time.
3. Use cards and KPIs to track performance metrics.
